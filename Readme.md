Just summarizing what's in the [official documentation](https://docs.platformio.org/en/latest/plus/debug-tools/stlink.html#drivers) for setting up the st-link

For windows install [ST's driver](https://www.st.com/en/development-tools/stsw-link009.html)

For linux install the [platformio udev rules](https://docs.platformio.org/en/latest/core/installation/udev-rules.html#platformio-udev-rules)

For mac setup should be automatic
